# Retro Pixel of Tomorrow

Proyecto web frontend para tienda de videojuegos retro y actuales.

## Instalación y ejecución

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia el servidor de desarrollo:
   ```bash
   npm start
   ```

## Estructura del proyecto
- `src/` - Código fuente principal
- `components/` - Componentes reutilizables
- `pages/` - Vistas principales (Home, Games, Consoles, Contact)
- `Context/` - Contextos globales (Carrito, Auth)
- `audio/` - Archivos de música
- `public/img/` - Imágenes y assets

## Estilo visual
- Fondo global con gradiente retro y píxeles flotantes, igual que la página de Contacto.
- Paleta de colores unificada: gris oscuro, acentos en amarillo, rosa, cian, verde y púrpura.
- Accesibilidad mejorada en menús, botones, formularios e imágenes.

## Mejoras implementadas
- Tema oscuro fijo y fondo animado retro en todo el sitio.
- Lazy loading de imágenes y páginas.
- Accesibilidad en navegación, formularios y controles.
- Optimización de performance en componentes clave.
